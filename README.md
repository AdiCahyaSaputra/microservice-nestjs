## Structure Overview

- Order Service
  Used as starting point for food order. Run `pnpm start:dev` at `order-service`

- Order Worker Service
  Used to monitor and update order status. Run `pnpm start:worker:dev` at `order-service`

- Kitchen Service
  Used to process the created order. Run `pnpm start:dev` at `kitchen-service`
